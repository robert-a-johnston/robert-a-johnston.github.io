Get to Know Me

Greetings! I'm Robert Johnston, a software developer with a unique journey that began in the world of zookeeping, where I spent nine enriching years. A career transition due to an injury led me to the captivating realm of software engineering, which, much like zookeeping, offers perpetual learning opportunities. After completing my degree, I sought to enhance my expertise, embarking on a transformative three-month immersive software engineering course at General Assembly.

For the past two years, I've thrived as a consultant at Accenture, collaborating with global giants across various sectors such as transportation, e-commerce, business services, entertainment, and aerospace and defense. During this time, I've pioneered numerous automation solutions and achieved certifications including AWS Solutions Architect and Developer Associate, as well as Azure Fundamentals and Terraform Associate.

My journey epitomizes adaptability and an unwavering commitment to continuous growth in the ever-evolving landscape of software development. I'm enthusiastic about confronting new challenges and leveraging my skills to drive meaningful change.

Capstone Project

A capstone project using React for the client and Django, Python, and a PostgreSQL database for the server. The purpose of the app is to allow the user to create a database of seed information. The user can sign in and create seeds and note if they are favorites or planted in the garden. The user can also update the information about the seeds as that information changes.

Message Board

The message board was a group project. The group used React for the client and Express, Mongoose, and a MongoDB database for the server. In this app, a user can sign in and create posts and comments for others to read. The user can also edit and delete those posts that they created but not posts created by other users. This was the first React framework full-stack application built.

Zoo Builder

Zoo Builder uses Javascript, HTML, and SCSS for the server and Express, Mongoose, and a MongoDB database for the server. In this app, a user can sign in and create a list of animals they would like to be in their zoo. The user can update animal information and remove animals from the zoo if they would like. This was the first full-stack application built.

Tic Tac Toe!

Tic-Tac-Toe uses JavaScript, HTML, and CSS to create a server-side tic-tac-toe game. This was the first project created using Git and GitHub as version control methods that would be used in all of the other projects. This project demonstrated the agile workflow that would be expected in all future projects.

Algorithm Fun

Research into different path searching and array sorting algorithms led to a project that has a visual representation of several different algorithms. Also provided is information about each algorithm as well as a short description of implementation. The implementation is one of multiple ways to implement each algorithm. The site is created using React with React hooks for the sorter and React classes for the path finder. It also contains HTML, CSS, bootstrap, and JavaScript.
